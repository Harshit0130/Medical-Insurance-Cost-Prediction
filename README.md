# 🏥 Medical Insurance Cost Prediction

## 📌 Problem Statement
Predict insurance charges based on demographic and lifestyle features such as age, BMI, smoking status, and region.

---

## ⚙️ Tech Stack
- Python
- Pandas, NumPy
- Scikit-learn
- Matplotlib, Seaborn

---

## 📊 Exploratory Data Analysis
Key insights:
- Smoking significantly increases insurance costs
- Higher BMI leads to higher charges
- Age is positively correlated with insurance expenses

---

## 🤖 Models Used
- Linear Regression
- Random Forest Regressor

---

## 📈 Results

| Model | R² Score | MAE |
|------|--------|--------|
| Linear Regression | 0.78 | ₹4213 |
| Random Forest | **0.86** | **₹2553** |

---

## 🧠 Key Learnings
- Random Forest performs better for non-linear relationships
- Feature engineering and EDA are crucial for performance
- Smoking and BMI are major cost drivers
